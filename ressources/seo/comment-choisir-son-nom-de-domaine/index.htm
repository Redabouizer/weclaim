Comment choisir son nom de domaine ?

14 mars 2023 – Roland Debrabant
Date de mise à jour : 12 août 2025

  • SEO

Vous cherchez quel nom de domaine réserver pour votre site internet ?

Le nom de domaine est l’équivalent de votre adresse, voire de votre nom de marque sur internet, puisque c’est ce que les internautes taperont sur leur moteur de recherche pour atteindre votre site, une fois que celui-ci sera connu. Le nom de domaine (ou NDD) joue donc un rôle essentiel dans une stratégie webmarketing. Ce nom définira votre identité sur le web et peut avoir un impact considérable sur votre SEO.

Voici donc les points à soulever venu le moment de se demander comment choisir un nom de domaine optimisé pour le référencement :

Qu’est-ce qu’un nom de domaine ?

Un nom de domaine est l’adresse unique qui permet d’accéder à votre site web sur Internet. C’est la version « lisible par un humain » de l’adresse IP utilisée par les serveurs pour communiquer.

Il se compose de deux éléments :

  • Le nom : la partie personnalisée choisie par le propriétaire (exemple)
  • L’extension (ou TLD – Top Level Domain) : la partie située après le point (.com, .fr, etc.)

Exemple : dans mon-site.fr :

  • mon-site est le nom
  • .fr est l’extension

Au-delà de sa fonction technique, le nom de domaine est un élément stratégique :

  • Il participe à votre identité de marque.
  • Il peut influencer la mémorisation par vos visiteurs.
  • Il joue un rôle dans la visibilité SEO et la perception de fiabilité par les internautes.

Choisir un nom de domaine ne se limite donc pas à « trouver quelque chose de libre » : c’est un investissement à long terme pour votre présence en ligne.
Avant de faire votre choix, faites un test de lisibilité et un test radio pour vérifier que votre nom de domaine se mémorise bien.

Le choix de l’extension / TLD

Avant même de définir l’objet principal de votre nom de domaine, il faut choisir l’extension (ou TLD, pour Top Level Domain).

Les extensions génériques les plus répandues sont :

  • .com : historiquement lié aux activités commerciales (aujourd’hui universel et le plus utilisé).

  • .net : initialement pour les sites orientés contenu ou réseaux.

  • .org : historiquement pour les organisations non lucratives.

Aujourd’hui, même si le .com est devenu la référence par défaut, ces trois extensions restent pertinentes, surtout si vous visez une stratégie flexible (langues, pays, thématiques).

Les TLD localisés

Certains TLD sont rattachés à un pays (.fr, .be, .ca, .pt…).
Ils offrent un léger avantage SEO dans leur pays respectif, mais peuvent limiter une stratégie internationale basée sur des répertoires linguistiques (/en, /fr, /es…).

En revanche, ils sont idéaux pour :

  • Renforcer un ancrage local (Made in France, proximité, terroir).

  • Une stratégie internationale multi-domaines avec un domaine par pays.

Exemple :

  • exemple.com (site principal)

  • exemple.fr

  • exemple.es

Les TLDs spéciaux/nouveaux TLDs

Depuis quelques années, de nouvelles extensions sont apparues :

  • Localités : .paris, .london

  • Secteurs d’activité : .auto, .shop

Selon Google : “Nous renvoyons les résultats les plus pertinents, quel que soit le domaine de premier niveau.”

En clair : le TLD seul n’améliorera pas votre SEO, mais un TLD aligné avec votre activité peut renforcer votre image de marque.
⚠️ Vérifiez juste qu’il ne limitera pas vos projets futurs (ex. expansion internationale).

Choisir un nom de domaine brandé ou EMD ?

Deux possibilités s’offrent ensuite à vous concernant le contenu même de votre nom de domaine : nom de marque ou mot-clef ?

D’abord, qu’est-ce qu’un EMD ?

Un EMD, ou Exact Match Domain, est un nom de domaine qui correspond à un ou des mots-clefs. Exemple : plombier-paris.com

Son impact sur le SEO est décrié depuis quelques années et il n’est plus autant plébiscité qu’auparavant. Même si son efficacité décline, il n’en reste pas moins un critère toujours viable dans une stratégie de référencement et doit être pris en compte.

Toute la question repose sur la concurrence de votre secteur et/ou de votre mot-clef principal, et sur les retombées prévisibles par rapport au préjudice subi par un nom de marque moins renforcé.

La combinaison des deux possibilités peut aussi parfaitement convenir et peut même être le combo gagnant pour une balance cohérente entre votre nom de marque et l’optimisation de votre SEO ! Exemple : agence-mannequin-onetop.com

Quoi qu’il arrive, évitez les noms de domaine trop long et d’autant plus que vous faites le choix d’y insérer des mots-clefs. Evitez aussi les noms de domaine avec accent

Les avantages d’un nom de domaine brandé

Un nom de domaine brandé est construit autour de votre marque plutôt que de mots-clés purs (exemple : leboncoin.fr, blablacar.com).

Il présente plusieurs avantages :

  • Mémorisation facile : un nom unique et distinctif s’imprime plus facilement dans l’esprit des internautes.

  • Polyvalence : vous n’êtes pas enfermé dans une thématique précise et pouvez faire évoluer votre activité sans changer de domaine.

  • Différenciation : vous vous démarquez des concurrents qui utilisent des noms descriptifs génériques.

  • Crédibilité : un nom inventé ou travaillé donne une image plus professionnelle et sérieuse.

  • Potentiel marketing : plus simple à décliner sur les supports de communication et les réseaux sociaux.

  • Protection juridique : plus facile à déposer à l’INPI et à protéger contre les copies.

En SEO, un nom brandé ne joue pas directement sur le positionnement via les mots-clés, mais il contribue à bâtir une notoriété de marque. Or, plus les internautes recherchent votre marque, plus Google perçoit votre site comme une référence fiable. Le branding est aussi une des clés du GEO.

Nom de domaine avec ou sans tirets ?

D’un point de vue purement SEO, les tirets n’ont pas réellement d’impact. Vous pouvez donc vous concentrer sur la lisibilité de votre nom de domaine. Un nom de domaine assez long aura tendance à être plus lisible en y mettant des tirets et plus facilement reconnaissable d’un autre, donc plus impactant pour les internautes. Exemple : plombier-paris-artisans.fr

Un nom de domaine plus court et/ou plus “brandé” par contre peut tout à fait se passer de tirets. Exemple : mypattoune.com

Ce choix a l’avantage de ne pas rajouter de difficulté à vos clients pour vous trouver directement quand ils vous cherchent, en tapant l’URL de votre site directement dans la barre de recherche par exemple.

En bref, les tirets, quand on peut s’en passer, à éviter, mais si on ne peut pas, ne pas hésiter !

Prix d’un nom de domaine

Petit point d’attention : les extensions n’ont pas toutes le même prix !

Un .com vous coûtera une dizaine d’euros tandis qu’un .fr sera entre 5 et 7€. Mais il existe des domaines biens plus chers notamment sur les nouveaux TLDs, comme le .auto se négociant autour des 2 200€

Il existe également un marché secondaire des noms de domaine, qui sont déjà détenus par un tiers et mis aux enchères directement chez les registrars ou sur d’autres plateformes. Vérifiez donc que votre nom de domaine soit disponible dans toutes les extensions que vous souhaitez obtenir avant de vous décider.

Acheter un nom de domaine expiré

Si vous n’êtes pas fermés sur un nom en particulier, choisir un nom de domaine expiré peut être intéressant selon l’historique du site, vous pourrez avoir un avantage en référencement en profitant de l’ancienneté du domaine.

Vérifiez bien les backlinks et les différentes métriques avant de valider ce choix, certains sites expirés ont été spamés ou peuvent ne pas correspondre exactement à votre thématique.

Vous pouvez même tenter de retrouver les anciens contenus du site sur Wayback Machine et les remettre en ligne ! Content is king

Comment procéder à l’enregistrement de son nom de domaine ?

L’enregistrement d’un nom de domaine se fait auprès d’un registrar (bureau d’enregistrement) accrédité par l’ICANN (pour les TLD génériques) ou par l’organisme national compétent (AFNIC pour le .fr). Voici les étapes clés :

  1. Choisir un registrar fiable
    Exemples : OVHcloud, Gandi…
    Privilégiez un acteur reconnu pour éviter les problèmes de gestion ou de transfert ultérieur.

  2. Vérifier la disponibilité
    Utilisez l’outil de recherche du registrar pour tester votre nom dans toutes les extensions que vous souhaitez.
    💡 Pensez à vérifier aussi les déclinaisons proches pour éviter le cybersquatting.

  3. Valider les aspects juridiques et branding

    • Vérifier à l’INPI l’absence de marque déposée similaire.

    • Vérifier la disponibilité sur les réseaux sociaux.

  4. Enregistrer pour plusieurs années
    C’est souvent plus économique et peut envoyer un signal de fiabilité à Google.

  5. Activer les options de protection

    • Whois Privacy : masque vos informations personnelles dans les bases publiques.

    • Verrouillage du domaine : empêche les transferts non autorisés.

  6. Configurer votre nom de domaine

    • Associer à un hébergement web.

    • Créer des adresses email professionnelles.

    • Mettre en place un certificat SSL (HTTPS).
