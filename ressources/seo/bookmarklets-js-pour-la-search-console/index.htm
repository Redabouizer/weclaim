Bookmarklets JS pour le SEO et la Search Console

21 octobre 2022 – David Groult
Date de mise à jour : 8 août 2025

  • SEO

Depuis avril 2021, la Search Console de Google offre la possibilité d’utiliser des regex (expressions régulières) pour trier les informations contenues dans le rapport de performances. Pour vous aider à mieux utiliser cette fonctionnalité complexe en apparence, nous allons vous fournir des bookmarklets JS à utiliser facilement avec votre navigateur pour gagner du temps et de la performance dans vos analyses SEO.

Les regex dans la Search Console

Tout d’abord, définissons ce qu’est une expression régulière (ou “regex”). En bref, il s’agit d’une séquence de caractères qui définit un motif de recherche. Vous pouvez utiliser les regex pour trouver des motifs spécifiques dans des chaînes de caractères ou du texte, par exemple pour trouver toutes les adresses électroniques ou URL dans un bloc de texte

Dans l’outil Google Search Console, vous pouvez désormais utiliser des regex dans le rapport de performance pour filtrer les données et n’afficher que des informations spécifiques. Cela peut être utile lorsque vous analysez une grande quantité de données ou lorsque vous essayez de trouver des modèles très spécifiques dans vos résultats de recherche.

Pour commencer à utiliser les regex, allez dans le rapport de performance et cliquez sur l’onglet “Pages”. À partir de là, vous verrez une petite flèche à côté de l’intitulé “Page”. Cliquez dessus et sélectionnez “Expression régulière” Vous pouvez ensuite taper votre expression régulière dans la barre de recherche

Les bookmarklets JS pour faciliter l’utilisation de ces Regex SEO

Pour aller plus vite encore dans l’exploitation des regex dans la Search Console, nous avons créé des bookmarklets JS.

Un bookmarklet JS est un petit morceau de code JavaScript que vous pouvez ajouter à la barre de favoris de votre navigateur (Chrome/Firefox/Edge…). Lorsque vous cliquez dessus, le code s’exécute et lance l’action souhaitée. Dans ce cas, nous l’utilisons pour exécuter rapidement une expression régulière sur le rapport Performance de la Search Console.

Pour ajouter le bookmarklet JS, il suffit d’ajouter les scripts suivants comme favori dans votre navigateur.

Une fois que vous les avez ajouté, vous pouvez simplement cliquer dessus chaque fois que vous voulez utiliser une regex sur une page. Il exécutera automatiquement la regex et filtrera les données dans le rapport de performance.

Identifier les mots-clés avec questions

javascript:(function()%7Bwindow.location.search%2B%3D'%26query=~'+encodeURIComponent('\\b(qui|quoi|que|quel|quels|quelle|quelles|qu.est.ce|ou|où|quand|comment|combien|pourquoi)+\\s')})()

Identifier les mots-clés avec questions (en anglais)

javascript:(function()%7Bwindow.location.search%2B%3D'%26query=~'+encodeURIComponent('\\b(are|can|how|what|when|where|which|who|why|will)+\\s')})()

Identifier les mots-clés de longue traine (> 4 termes)

javascript:(function()%7Bwindow.location.search%2B%3D'%26query=~'+encodeURIComponent('(\\w+\\s){4,}\\w+')})()

Identifier les requêtes avec une année

javascript:(function()%7Bwindow.location.search%2B%3D'%26query=~'+encodeURIComponent('\\b(20[0-9]{2})+\\s')})()

Identifier les requêtes avec une couleur

javascript:(function()%7Bwindow.location.search%2B%3D'%26query=~'+encodeURIComponent('\\b(rouge|bleu|blanc|noir|gris|jaune|violet|vert|orange|rose|marron|beige)+\\s')})()
