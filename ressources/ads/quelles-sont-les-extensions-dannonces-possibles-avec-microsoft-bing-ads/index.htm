Quelles sont les extensions disponibles sur Bing Ads

Une multitude d’extensions existent, nous vous conseillons de toutes les mettre en place afin d’optimiser votre compte. Cela permettra de rendre plus visible vos annonces et d’obtenir un meilleur quality score de votre compte

Pour ajouter des extensions, il suffit de vous rendre sur votre compte Microsoft Bing Ads, de sélectionner Annonces et Extensions puis de cliquer sur extensions :

Extension d’action

Cette extension permet l’ajout d’un bouton d’appel à l’action sur vos annonces. Cette extension existe dans plusieurs langues dont le français. Il existe un large panel de texte d’action (acheter, afficher la démo, allez-y, devis gratuit, voir l’offre…).

Le nombre de texte a évolué à la hausse ces derniers mois pour atteindre 60 types de textes différents.

L’extension peut s’ajouter au niveau du Compte / Campagne / Groupe d’annonces

L’extension d’application

L’extension d’application va vous permettre de mettre de booster les téléchargements de votre application. Vous pouvez ajouter une application Android / IOS / Windows phone / Windows.

Extension d’appel téléphonique

Cette extension permet d’afficher son numéro de téléphone qui sera cliquable sur mobile. Il est possible de modifier les horaires de diffusion afin que le service qui reçoit les appels puisse répondre dans les heures d’ouverture.

Extensions d’accroches

Cette extension vous permettra d’ajouter des éléments forts sur votre sociétés tel que : Livraison gratuite, Estimation gratuite, Réponse en 24h…

Le nombre de caractères est limité à 25, il faut donc choisir des arguments courts et incisifs.

Extension de lien de filtre

Cette extension vous permet de mettre en avant des catégories de produits ou services dont vous souhaitez mettre en avant.

Extensions d’images

Cette extension va s’afficher sur les résultats mobiles ou desktop. Depuis 2021, l’extension semble être de plus en plus visible (en réponse à l’intégration par Google de cette même extension). Cela permet de rendre un affichage encore plus visible face aux concurrents :

Voici les formats demandés par Bing :

  • format d’image 1.78 : 1 (Extension à image unique Bing Desktop)
  • format d’image 1,33 : 1 (Extension multi-image Bing Desktop)
  • format d’image 1,91 : 1 (Extension multi-image Bing Mobile)
  • format d’image 1.72 : 1 (Image large du Microsoft Audience Network)
  • format d’image 1,69 : 1 (Carrousel unique de Microsoft Edge)
  • format d’image 1,55 : 1 (Titre moyen dans Microsoft Edge)
  • format d’image 1,5 : 1 (Annonce dans un article MSN)
  • format d’image 1 : 1 (Image carrée du Microsoft Audience Network)
  • format d’image 0,93 : 1 (Double carrousel MSN)

Selon le format, l’emplacement diffère, il est donc nécessaire de fournir des visuels de différentes tailles pour que ceux-ci soient pris en compte.

Extension d’emplacements

Cette extension vous permet d’afficher l’adresse de votre entreprise, cela est très utile si vous êtes amenés à recevoir des clients dans vos locaux. Renseignez les différents champs et l’extension apparaitra sur mobile et desktop.

Extension de prix

Cette extension va vous permettre de mettre en avant différents prix. Il est nécessaire que les prix indiqués dans l’extension soit visible sur la page de destination. Cela vous permet de mettre en avant vos meilleurs produits.

Extension de promotion

Cette extension est très proche de l’extension de prix, elle va vous permettre de mettre en avant une promotion en fonction d’un événement.

Les avantages de cette extension sont de pouvoir programmer une date de début et de fin, d’avoir un événement mis en avant dans l’annonce, de pouvoir choisir un montant de remise, un pourcentage de réduction.

Extensions d’avis

Il y a deux formats de cette extension : une que l’on paramètre et l’autre qui s’ajoute automatiquement. La première est une critique dont la source doit provenir d’une source fiable. Il faut copier-coller un morceau de l’avis (il est interdit de le modifier) et d’ajouter l’URL de la critique. L’avis ne doit pas dater de plus d’un an.

Et pour l’autre extension elle est automatique et liée à votre site e-commerce. L’affichage apparait avec les étoiles et la note de votre site e-commerce.

Extensions de liens annexes

C’est l’extension de base qui permet d’ajouter un complément d’information avec un titre, deux lignes de description et une url de destination. Elle apporte une visibilité importante lors de l’affichage avec les lignes de descriptions. Les limites de caractères sont les suivantes : 35 caractères que ce soit le titre ou par ligne de description.

Extensions de liens structurés

C’est une des extensions les plus récentes, elle permet d’afficher plus d’informations sur un aspect précis des produits & services mis en avant. Il ne faut pas que cela reprenne ce qui est déjà dans l’annonce ou dans l’extension d’accroche. Sur Google Ads, elle s’appelle Extrait de site, elle va vous permettre d’avoir une en-tête précise (à sélectionner parmi la liste proposée).

Extensions vidéos

Et enfin le dernier format d’extension qui va s’afficher sur desktop et mobile dans les résultats search. Nous recommandons d’utiliser ce format dans une campagne qui n’a pas d’extensions d’action, d’image ou d’appels téléphoniques afin de lui laisser toute la place.
